Fetch from joke api

You'll have to fetch some data from the joke Api. Display 10 random jokes in the table when you load the page. If you go to the documentation you'll see that you can fetch 10 random jokes of the general type or 10 random jokes of the programming type. Fetch 10 of both types and put 5 of each in an array. Display those in the table and sort those 10 jokes in the table based on the first character in the setup string alphabetically. The jokes that have type "programming" should have a black background and white font color.

After that when you click on the button after selecting a joke type, add a random joke from that type to the bottom of the table and if it's of the type "programming" make the background black and font color white again.
